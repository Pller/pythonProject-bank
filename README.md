# 🏦 Анализ банковских транзакций

Python приложение для анализа и визуализации банковских транзакций.

## 🎯 Цель проекта

Создание инструмента для анализа персональных финансов с возможностью:
- Анализа расходов по категориям
- Отслеживания кешбэка и бонусов
- Генерации отчетов
- Поиска специфических транзакций

## 🚀 Быстрый старт

### Установка зависимостей
\\\ash
pip install -r requirements.txt
\\\

### Запуск приложения
\\\ash
python -m src.main
\\\

### Запуск тестов
\\\ash
pytest tests/ -v
\\\

## 📁 Структура проекта

\\\
.
├── data/              # Папка с данными (Excel файлы)
├── src/               # Исходный код приложения
│   ├── main.py        # Главный скрипт
│   ├── views.py       # Веб-представления
│   ├── services.py    # Бизнес-логика
│   ├── reports.py     # Генерация отчетов
│   └── utils.py       # Вспомогательные функции
├── tests/             # Тесты
├── requirements.txt   # Зависимости Python
├── .flake8           # Настройки линтера
├── pyproject.toml    # Конфигурация проекта
└── README.md         # Документация
\\\

## 🔧 Технологии

- **Python 3.13** - основной язык
- **pandas** - обработка данных
- **pytest** - тестирование
- **flake8, black, isort** - проверка качества кода

## 📊 Функциональность

### ✅ Реализовано
- Загрузка данных из Excel
- Анализ категорий расходов
- Расчет кешбэка и бонусов
- Генерация JSON отчетов
- Поиск транзакций

### 🔄 В процессе
- Веб-интерфейс
- Графическая визуализация
- Экспорт отчетов в PDF

## 🧪 Тестирование

Проект включает модульные тесты:

\\\ash
# Запуск всех тестов
pytest

# С проверкой покрытия
pytest --cov=src tests/
\\\

## 📈 Качество кода

Проект использует инструменты для поддержания качества:

\\\ash
# Форматирование кода
black src/ tests/

# Сортировка импортов
isort src/ tests/

# Проверка стиля
flake8 src/ tests/
\\\
